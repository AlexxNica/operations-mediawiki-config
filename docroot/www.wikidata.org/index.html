Welcome to Wikidata!

Wikidata aims to create a free knowledge base about the world that can be read and edited by humans and machines alike. It will provide data in all the languages of the Wikimedia projects, and allow for the central access to data in a similar vein as Wikimedia Commons does for multimedia files. Wikidata is currently in development so you unfortunately can't see the system running here yet. This will change soon.

In the meantime you can learn more about the project and participate.
